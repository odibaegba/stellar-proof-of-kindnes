# stellar-proof-of-kindnes
A decentralized gratitude and reputation protocol built on Stellar Soroban that allows users to issue verifiable appreciation records on-chain.
This project allows users to issue verifiable appreciation records on-chain
to recognize contributions, mentorship, and support within communities.

## Vision

The internet rewards attention and controversy.

This protocol rewards kindness and contribution.

## Built With

- Stellar
- Soroban Smart Contracts
- Rust
- Next.js
- Stellar SDK
